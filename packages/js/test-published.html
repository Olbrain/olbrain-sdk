🚀 Installation & Usage

1
npm Install
Install from npm registry
bash
npm install @olbrain/js-sdk
2
Browser Widget (One-Line)
Add AI chat to any website instantly
html
<script src="https://unpkg.com/@olbrain/js-sdk@latest/dist/widget.widget.global.js"></script> <script> new OlbrainWidget.ChatWidget({ agentId: 'your-agent-id', apiKey: 'sk_live_your_key' }).mount(); </script>
3
Node.js Usage
Use in server-side applications
javascript
import { AgentClient } from '@olbrain/js-sdk'; const client = new AgentClient({ agentId: 'your-agent-id', apiKey: 'sk_live_your_key' }); const sessionId = await client.createSession(); const response = await client.sendAndWait(sessionId, 'Hello!'); console.log(response.text);
4
TypeScript Support
Full type definitions included
typescript
import { AgentClient, ChatResponse } from '@olbrain/js-sdk'; const client = new AgentClient({...}); const response: ChatResponse = await client.sendAndWait(sessionId, 'Hi!');
