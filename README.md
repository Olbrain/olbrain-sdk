# Olbrain SDK

Official SDKs for [Olbrain](https://olbrain.com) Agent Cloud.

## Packages

| Package | Language | Path |
|---------|----------|------|
| [olbrain-python-sdk](packages/python/) | Python | `packages/python/` |
| [@olbrain/js-sdk](packages/js/) | JavaScript / TypeScript | `packages/js/` |

## Getting Started

### Python SDK

```bash
pip install olbrain-python-sdk
```

See the [Python SDK README](packages/python/README.md) for full documentation.

### JavaScript SDK

```bash
npm install @olbrain/js-sdk
```

See the [JavaScript SDK README](packages/js/README.md) for full documentation.
